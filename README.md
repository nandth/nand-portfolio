# abusaad-test-website

A simple barebones React website that displays "Hello, i am abusaad" with 2 React elements.

## Features
- Main heading: "Hello, i am Nand"
- 2 distinct React elements with styled content
- Built with React and Vite

## Getting Started

### Installation
```bash
npm install
```

### Development
```bash
npm run dev
```

### Build for Production
```bash
npm run build
```

### Preview Production Build
```bash
npm run preview
```
